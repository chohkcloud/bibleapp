│   ├── services/               # 비즈니스 로직
│   │   └── database/
│   ├── hooks/                  # 커스텀 훅
│   ├── store/                  # 상태 관리 (Zustand)
│   ├── utils/                  # 유틸리티

    ├── services/

| react-native-safe-area-context | Safe Area |

| @react-navigation/native | 네비게이션 코어 |
